CineMate | MERN Stack May 2024 - July 2024

• Developed a full-stack movie and TV show recommendation web application with a secure user registration and
login system for personalized access.

• Implemented user authentication and authorization using JWT (JSON Web Token) to ensure secure login and
access control.

• Applied role-based access control (RBAC) to distinguish between user and admin functionalities, with an admin
panel for managing categories and movies.

• Integrated a wishlist feature for users to save and access favorite movies along with detailed information such as
summaries and ratings.

• Developed a RESTful API to facilitate CRUD operations for users, movies, and categories, enhancing overall data
management.
